Bienvenido a Mi Sitio Web

Esta es una página web básica creada con HTML5 y CSS3 para demostrar el uso de elementos fundamentales del desarrollo web.

Este proyecto incluye todos los elementos solicitados: encabezados, párrafos, listas, enlaces internos y externos, y estilos CSS básicos.

Explora las diferentes secciones usando el menú de navegación superior.
